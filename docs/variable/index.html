    import Botao from 'projeto-eletrica-dimencionamento/src/Components/Botao/Botao.js'

    Styled-componente de botão estilizável com animação de mover 2px no eixo Y quando clicado e aumenta escala em 1.05 quando mouse esta em cima.

    Properties:

    NameTypeAttributeDescription
    props object

    Props React
    props.color string

    Define a cor da fonte. Valor padrão 'black'.
    props.backgroundcolor string

    Define a cor de fundo do componente.Valor padrão 'white'.
    props.border_radius string

    Define o raio de curvatura dos cantos do componente. Valor padrão '0px'.
    props.width string

    Define o comprimento do componente. Valor padrão '100%'.
    props.border string

    Define o estilo da borda do componente. Valor padrão '0px'.
    props.padding string

    Define o espaçamento interno do componente. Valor padrão '0px 0px'.
    props.margin string

    Define o espaçamento externo do componente. Valor padrão '0px 0px'.
    props.font_size string

    Define o tamanho da fonte. Valor padrão '14px'.
    props.boxshadow string

    Define a sombra do componente. Valor padrão '0px 0px 0px'.
    props.colorHover string

    Define a cor do texto quando o mouse esta em cima do componente.

    Example:

    <Botao color='orange' backgroundcolor='blue' border='1px solid black' border_radius='2.5px' />
